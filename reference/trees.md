# Trees dataset

A dataset used throughout rsgl documentation and examples containing
repeated measurements of tree growth over time. Loading rsgl masks
[datasets::trees](https://rdrr.io/r/datasets/trees.html); refer to the
built-in version as
[`datasets::trees`](https://rdrr.io/r/datasets/trees.html) if you need
it.

## Usage

``` r
trees
```

## Format

A data frame with 35 rows and 3 variables:

- tree_id:

  Integer identifier for the tree.

- age:

  Age of the tree (in days) at the time of measurement.

- circumference:

  Trunk circumference at the time of measurement.
