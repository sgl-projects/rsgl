# rsgl

R package implementing the SGL graphics language. Derived from the
sglref reference implementation.

## Syncing with sglref

rsgl tracks the sglref reference implementation at
`/Users/jochapjo/sgl_projects/sglref/`. To sync with latest sglref
changes, use `/project:sync-sglref`.

The file `sglref-sync.txt` records the last synced sglref commit hash.
